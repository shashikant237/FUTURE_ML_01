# 📈 Sales & Demand Forecasting for Businesses

## 📌 Future Interns - Machine Learning Internship (Task 1)

This project was developed as part of the **Future Interns Machine Learning Internship**.

The objective is to build a Machine Learning model capable of forecasting future business sales using historical sales data.

---

## 🚀 Project Objective

Predict future sales using historical business data and provide business-friendly insights through data visualization and forecasting.

---

## 📂 Dataset

Dataset Used:
Sample - Superstore Dataset

Contains:

- Orders
- Sales
- Profit
- Discount
- Quantity
- Product Information
- Customer Information
- Region
- Order Date
